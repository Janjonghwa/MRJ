# 신규 유저 온보딩 및 UX 플로우 명세서

## 1. 개요
* **문서 목적:** 신규 유저가 앱에 진입하여 초개인화 데이터(생년월일, MBTI)를 입력하고, 사주 분석 결과를 확인한 뒤 지인에게 공유하기까지의 핵심 여정(User Journey)을 정의합니다.
* **UX/디자인 핵심 원칙:** 
  - 딱딱한 폼(Form) 형식을 배제하고, 하나의 화면에 하나의 질문만 띄우는 **대화형 스텝퍼(Stepper)** 형식 채택.
  - 여백의 미(White space)와 자연스러운 페이드 트랜지션을 극대화하여 **'디지털 한옥' 특유의 정적이고 고급스러운 분위기** 연출.

---

## 2. 화면별 플로우 상세 (Screen-by-Screen Flow)

### 2.1 랜딩 페이지 (Landing Page)
* **목표:** 서비스의 프리미엄 분위기를 전달하고 '입력 시작'을 유도.
* **UI 요소:**
  - **Hero 타이틀:** "당신의 사주와 MBTI가 그리는 단 하나의 이야기" (가제)
  - **배경:** 은은하게 번지는 수묵화(먹물) 애니메이션 혹은 단정한 한지 질감의 미니멀 배경.
  - **CTA 버튼:** [내 운명 분석하기] (동양적 문양의 호버 트랜지션 적용)
* **동작 로직:** CTA 클릭 시 화면 전체가 부드러운 검은색/짙은 먹색으로 페이드되며 입력 스텝퍼로 전환.

### 2.2 정보 입력 폼 (Input Stepper)
인지 부하를 줄이기 위해 1 Screen 1 Question 원칙 적용. 로컬 스토리지에 임시 저장 기능을 두어 중간 이탈을 방지.

* **Step 1: 이름** 
  - 텍스트: "어떻게 불러드리면 될까요?" 
  - 액션: 이름 입력 후 자동 포커싱, Enter로 다음 화면 이동.
* **Step 2: 생년월일** 
  - 텍스트: "태어난 날짜를 알려주세요. (양/음력)" 
  - 액션: 네이티브 달력보다는, 연/월/일이 직관적으로 분리된 드롭다운이나 부드러운 슬라이더 형태 권장.
* **Step 3: 태어난 시간** 
  - 텍스트: "태어난 정확한 시간을 아시나요?"
  - 액션: 시간 선택 드롭다운 + [정확히 모름] 옵션 필수 제공 (명리학에서 시간 모름 처리에 대비).
* **Step 4: MBTI** 
  - 텍스트: "A님의 평소 성향은 어떠신가요?" 
  - 액션: 16개 성격 유형을 나타내는 감성적인 아이콘/버튼 그리드, 클릭 시 즉시 다음 화면으로.
* **Step 5: 핵심 질문 (관심사)** 
  - 텍스트: "지금 A님이 가장 궁금한 것은 무엇인가요?"
  - 액션: [올해의 종합 운세] / [나의 찰떡 직업운] / [평생의 연애 및 배우자운] 중 택 1. 선택 시 로딩 화면 진입.

### 2.3 분석 및 로딩 페이지 (Loading Screen)
* **목표:** 자체 만세력 분석 및 LLM(AI) 텍스트 생성에 걸리는 시간(약 3~7초) 동안 유저의 이탈을 막고 신비감을 고조시킴.
* **UI 요소:**
  - **중앙 애니메이션:** 전통적인 윤도(나침반)가 천천히 돌아가거나, 오행(목화토금수) 심볼이 반짝이며 교차하는 인터랙션.
  - **진행 상황 멘트 (2초 간격 롤링):**
    1. "만세력 명식을 추출하고 있습니다..."
    2. "사주 오행과 MBTI 유형을 교차 분석 중입니다..."
    3. "과거와 성향을 바탕으로 운명의 서사를 작성 중입니다..."

### 2.4 결과 및 요약 페이지 (Result & Share Card)
* **목표:** 무료 결과의 질적 가치를 극대화하여 보여주고, 본인의 개성 넘치는 결과 카드를 공유하도록 강력히 유도.
* **UI 요소:**
  1. **헤더 카피 (LLM 생성):** "타오르는 불을 품은 겨울 숲, 그리고 냉철한 전략가(INTJ)" 같은 시적이고 함축적인 메인 타이틀.
  2. **핵심 요약 (무료):**
     - 나를 상징하는 3가지 오행/동물 아이콘.
     - 성격, 강점, 조심해야 할 점에 대해 사주와 MBTI가 결합된 부드러운 풀이 단락 제공.
  3. **바이럴 공유 썸네일 (Shareable Card):** 
     - 인스타그램 스토리 사이즈(9:16)에 특화된 시각적 요약 카드. (예: "저를 설명하는 오행은 [물]입니다.")
     - 버튼: [결과 이미지 다운로드] / [카카오톡 공유하기]
  4. **심화 리포트 티저 (페이월 - Phase 2 대비):** 
     - 스크롤 하단에 "2026년 상세 월별 흐름", "나의 평생 배우자 몽타쥬 확정판"과 같은 타이틀을 블러(Blur) 이미지와 노출하여 프리미엄 과금 유도.
  5. **궁합 CTA (핵심 Viral Hook):** 👉 [친구들에게 나침반 보내기 (얼마나 잘 맞는지 궁합 보기)]

### 2.5 수신자 유입 플로우 (Receiver UX - Referral)
* **상황:** A유저가 B유저(지인)에게 카카오톡 혹은 인스타그램 스토리를 통해 '궁합 훅 링크'를 공유함.
* **진입 (Landing for B):** "A님과 당신의 운명적인 궁합은 몇 점일까요? 지금 사주와 MBTI로 확인해 보세요."
* **동작:** B유저도 호기심에 이끌려 [2.2 정보 입력 폼]을 수행하게 되며, 입력 완료 시 단순 본인의 사주 결과 + **"A님과의 다각도 궁합 풀이 특별 섹션"**이 함께 제공됨. (가장 강력한 유저 Lock-in 및 바이럴 루프).
